# heallows06 — Minimal Next.js

Ultra-clean Next.js starter for Vercel.

## Run
```bash
npm i
npm run dev
# http://localhost:3000
```

## Health checks
- JSON: `/api/health`
- Plain page: `/plain`

## Deploy (repo root)
- Put these files at the **root** of your repository.
- In Vercel: Import repo → Deploy (Framework: Next.js). 
- Leave **Root Directory** blank.
